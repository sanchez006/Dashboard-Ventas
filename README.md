# PAGINA VENTAS

Proyecto para la gestión de ventas y dashboard.

Estructura principal:
- `backend/` – API en TypeScript/Node
- `frontend/` – aplicación Angular
- scripts Python en la raíz

Instrucciones rápidas:
1. Instala dependencias en `backend` y `frontend` según sus `package.json`.
2. Configura variables de entorno en `backend/config/env.ts`.
3. Ejecuta el backend y frontend por separado (ver `backend/README.md`).
